Game Rules
1. 2D platform game
2. Collect all cherries in the level to win the game
3. Only 5 lives per game
4. Deduct player's live by 1 when attacked by the enemy

-----------------------------------------
* Keyboard
- Left : Move to the left
- Right : Move to the right
- Up : Climb
- Down : Crouch
- Spacebar : Jump

* Mouse
- Left click : Click the buttons on the screen
